Overview

The Fine Print is a recently opened bookstore based in Montclair, NJ that operates online. In hopes to expand their business and provide better service to their customers, The Fine Print has decided to utilize a database.

Each page of this website allows employees to insert and view entries for that page's table. In addition, orders can be updated, products can be searched, and coupons can be deleted - each of these functionalities can be done on their corresponding page.

Each table in our schema has an associated page.

  • Orders
    • Order ID
    • Order Date
    • Number of Products
    • Total Cost
    • Customer ID
    • Coupon ID
    • The following functionalities are available: Select, Insert, Update
  • Products
    • Product ID
    • Item Name
    • Price
    • Quantity Available
    • The following functionalities are available: Select, Insert, Search
  • Orders/Products
    • ID
    • Order ID
    • Product ID
    • The following functionalities are available: Select, Insert
  • Customers
    • Customer ID
    • First Name
    • Last Name
    • Sign Up Date
    • Birthdate
    • The following functionalities are available: Select, Insert
  • Coupons
    • Coupon ID
    • Promotion Name
    • Percent Off
    • The following functionalities are available: Select, Insert, Delete
  • Coupons/Customers
    • Coupon Promotion
    • Customer Name
    • The following functionalities are available: Select, Insert
